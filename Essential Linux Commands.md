# Essential Linux Commands

# 1. File and Directory Management

О‚· **ls** Бқ“ List directory contents
О‚· **cd** Бқ“ Change directory
О‚· **pwd** Бқ“ Print working directory
О‚· **cp** Бқ“ Copy files and directories
О‚· **mv** Бқ“ Move or rename files and directories
О‚· **rm** Бқ“ Remove files or directories
О‚· **mkdir** Бқ“ Make directories
О‚· **rmdir** Бқ“ Remove empty directories
О‚· **touch** Бқ“ Change file timestamps or create empty files
О‚· **find** Бқ“ Search for files in a directory hierarchy
О‚· **locate** Бқ“ Find files by name
О‚· **tree** Бқ“ Display directories in a tree-like format
О‚· **chmod** Бқ“ Change file permissions
О‚· **chown** Бқ“ Change file owner and group
О‚· **chgrp** Бқ“ Change group ownership
О‚· **stat** Бқ“ Display file or file system status
